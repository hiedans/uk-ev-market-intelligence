# UK EV Market Intelligence

## Project Overview

This project analyses the UK automotive market with a focus on:

- ZEV Mandate impact
- Battery Electric Vehicle (BEV) adoption
- Emerging Chinese automotive brands
- Market forecasting
- Competitive intelligence

## Business Problem

The UK automotive industry is undergoing rapid transformation due to:

- Government ZEV regulations
- Electrification trends
- New market entrants
- Changing consumer behaviour

This project provides data-driven insights to support strategic decision-making.
